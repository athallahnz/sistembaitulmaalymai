## âœ¨ About the Application

The **Baitul Maal Financial Information System** is a web-based application developed to support foundations in:

- Managing **infaq, shadaqah, zakat, and social funds**
- Recording transactions using **double-entry accounting**
- Providing **real-time and accurate financial reports**
- Ensuring financial transparency for administrators and congregants

This system is specifically designed for **religious institutions and social foundations**, offering a simple yet structured workflow aligned with accepted accounting standards.
